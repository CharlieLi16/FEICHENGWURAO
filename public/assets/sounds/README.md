# 🔊 音效素材 Skeleton

> **Skeleton 模式**: 代码已预设好所有音效触发点，你只需要按照下表的文件名上传音频文件即可！

## 快速开始

1. 准备音频文件（推荐 MP3 格式）
2. 按照下表重命名文件
3. 放入此文件夹 (`public/assets/sounds/`)
4. 刷新页面，音效自动生效！

## 🎧 DJ 控制台

在导演控制台 (`/director`) 有一个 DJ 音效台，可以：

- **实时调节音量** - 滑块调节，支持预设 25%/50%/75%/100%
- **手动触发任意音效** - 点击按钮即可播放
- **音量影响所有音效** - 包括自动触发的音效

---

## 📁 支持格式

- `.mp3` ✅ 推荐（兼容性最好）
- `.wav` ✅ 支持
- `.ogg` ✅ 支持
- `.m4a` ⚠️ 部分浏览器支持

---

## ⚠️ 注意事项

1. **文件名区分大小写** - 必须与表格完全一致
2. **文件大小** - 建议 < 500KB（加载更快）
3. **音量一致性** - 建议所有音效的响度保持一致
4. **缺失文件不会报错** - 系统会静默跳过不存在的音效文件
