# VCR 视频文件夹

在这里放置男嘉宾的 VCR 视频文件。

## 建议的文件命名

| 文件名 | 用途 |
|--------|------|
| `1-vcr1.mp4` | 1号嘉宾 - 基本资料 |
| `1-vcr2.mp4` | 1号嘉宾 - 情感经历 |
| `2-vcr1.mp4` | 2号嘉宾 - 基本资料 |
| `2-vcr2.mp4` | 2号嘉宾 - 情感经历 |
| ... | ... |
| `6-vcr2.mp4` | 6号嘉宾 - 情感经历 |

或者用嘉宾名字：
| `张三-vcr1.mp4` | 张三 - 基本资料 |
| `张三-vcr2.mp4` | 张三 - 情感经历 |

## 在 /director/setup 中填写 URL

```
VCR1 URL: /assets/videos/1-vcr1.mp4
VCR2 URL: /assets/videos/1-vcr2.mp4
```

## 支持格式

- `.mp4` (推荐)
- `.webm`
- `.mov`

## 注意事项

- 每个 VCR 建议 2 分钟以内
- 分辨率建议 1080p
- 文件大小建议 < 100MB（Vercel 限制）
