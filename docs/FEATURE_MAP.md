## 数据结构

### 女嘉宾 FemaleGuest
```
id, name, nickname, age, constellation, school, major, 
photos[], tags[3], introduction
```

### 男嘉宾 MaleGuest
```
id, name, nickname, age, school, major, photo, 
vcr1Url, vcr2Url, introduction, question
```

### 幻灯片 SlideSlot
```
id (opening/intermission/success/fail/ending/custom-1~5), 
name, imageUrl
```

## 添加新功能指南

1. **在上表 Feature Request 列写入需求**
2. 描述清楚：
   - 在哪个环节触发？
   - 舞台上显示什么？

### 示例格式

```

在 qa1 环节，导演台增加一个"显示提问"按钮，
点击后舞台中央显示一个卡片，内容是预设的问题列表。
```
