资源文件夹说明
================

请将您的设计资源放入以下文件夹：

📁 icons/    - 图标文件 (SVG, PNG, ICO 等)
📁 images/   - 图片文件 (JPG, PNG, WebP 等)

使用方式：
- 在代码中引用: /assets/icons/your-icon.svg
- 在代码中引用: /assets/images/your-image.png

例如：
<img src="/assets/icons/logo.svg" />
<img src="/assets/images/banner.jpg" />

推荐格式：
- 图标: SVG (矢量，推荐) 或 PNG
- 图片: WebP (体积小) 或 JPG/PNG
- Favicon: favicon.ico 或 favicon.svg
