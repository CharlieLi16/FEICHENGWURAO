## 音效文件对照表

## 支持格式

- `.mp3` (推荐)
- `.wav`
- `.ogg`

## 注意事项

- 文件大小建议 < 500KB
- 音效时长建议 1-5 秒
- 确保音量一致
- 文件名区分大小写！
